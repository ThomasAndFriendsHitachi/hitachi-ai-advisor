## Important Classes Reference

```tsx
// Layout
min-h-screen flex items-center justify-center
// Full viewport height, flex centering

w-full max-w-md
// Responsive width (full on mobile, max 448px on desktop)

px-4 py-8
// Responsive padding (16px horizontal, 32px vertical)

p-8
// Card internal padding

space-y-6
// Vertical gap between sections (24px)

// Colors
bg-[#003366]        // Hitachi Blue (primary)
bg-[#004a7f]        // Hitachi Blue (hover)
bg-slate-50         // Page background
text-slate-900      // Dark text
border-slate-200    // Light border

// Effects
shadow-lg           // Large shadow
rounded-xl          // 12px border radius
border border-slate-200  // 1px gray border

// Text
text-2xl font-bold  // Title (24px, bold)
text-sm             // Subtitle (14px)
text-xs             // Helper text (12px)
font-semibold       // Button text (600 weight)
```

## Test Cases

### ✅ SSO Path
1. See login page
2. Click "Sign In with Hitachi SSO"
3. Button shows "Connecting..."
4. After 1 second, app shows dashboard

### ✅ Email/Password Path
1. Click "Use Email & Password"  
2. Form appears with email and password fields
3. Try to submit with empty fields
   → Error: "Please enter both email and password"
4. Enter "invalidmail" (no @)
   → Error: "Please enter a valid email address"
5. Enter "user@example.com" + "password"
   → Success (after 0.8s delay)

### ✅ Back to SSO
1. On email/password form
2. Click "Back to SSO"
3. Form disappears, returns to initial state
4. Email/password fields are cleared

### ✅ Help Link
1. Click "Need help? Contact your system administrator"
2. Should navigate (currently # placeholder)
3. Text color changes on hover (slate-500 → slate-700)
